SDK가 제공하는 기능

LG Innotek 카메라 모듈 SDK는 이미지 캡처부터 실시간 영상 처리까지 카메라 애플리케이션 개발에 필요한 모든 기능을 제공합니다.

📷

이미지 캡처 & 스트리밍

고해상도 이미지 캡처 및 실시간 영상 스트리밍을 위한 완전한 API 세트. RAW, JPEG, YUV 등 다양한 포맷을 지원합니다.

🎛️

카메라 제어

노출, 화이트 밸런스, ISO, 초점, 줌 등 카메라 파라미터를 세밀하게 제어할 수 있는 풍부한 인터페이스를 제공합니다.

⚡

하드웨어 가속 ISP

LG Innotek 카메라 모듈 내장 ISP(Image Signal Processor)를 활용하여 실시간 고품질 이미지 처리를 수행합니다.

🔭

멀티 카메라 지원

복수의 카메라 모듈을 동시에 제어하고 동기화할 수 있는 멀티 카메라 세션 관리 기능을 제공합니다.

🤖

AI / 머신러닝 통합

객체 감지, 얼굴 인식, 장면 분류 등 AI 기반 이미징 기능과의 손쉬운 통합을 위한 인터페이스를 제공합니다.

🔒

보안 및 인증

카메라 모듈의 보안 부팅 및 펌웨어 인증을 통해 신뢰할 수 있는 안전한 이미징 환경을 보장합니다.

SDK 구성

SDK 모듈 구성요소

각 모듈은 독립적으로 사용하거나 조합하여 최적의 이미징 솔루션을 구성할 수 있습니다.

Camera Core Library

v3.2.1

카메라 초기화, 세션 관리, 기본 캡처 파이프라인을 제공하는 핵심 라이브러리입니다. C/C++ 및 Java 바인딩을 지원합니다.

ISP Control API

v2.8.0

하드웨어 ISP 파라미터 직접 제어를 위한 저수준 API. AWB, AE, AF 알고리즘의 세밀한 튜닝이 가능합니다.

Image Processing SDK

v2.5.3

노이즈 제거, HDR 합성, 디모자이킹 등 고급 이미지 처리 알고리즘 라이브러리입니다.

Streaming Engine

v1.9.2

RTSP/HLS 프로토콜을 통한 실시간 영상 스트리밍 및 H.264/H.265 하드웨어 인코딩을 지원합니다.

Calibration Toolkit

v1.4.0

렌즈 왜곡 보정, 색상 교정, 카메라 교정 데이터 관리를 위한 도구 모음입니다.

Diagnostic & Logger

v1.2.1

카메라 모듈 상태 모니터링, 오류 진단 및 상세 로깅 기능을 제공하는 진단 모듈입니다.

호환성

지원 플랫폼

다양한 운영체제 및 임베디드 환경에서 동일한 SDK API로 개발할 수 있습니다.

빠른 시작

시작하는 방법

단 4단계로 LG Innotek 카메라 모듈 SDK를 여러분의 프로젝트에 통합하세요.

1

SDK 다운로드

개발자 포털에서 플랫폼에 맞는 SDK 패키지를 다운로드합니다.

2

라이브러리 설치

패키지 관리자 또는 수동 설치를 통해 SDK 라이브러리를 프로젝트에 추가합니다.

3

초기화 및 설정

API 키 발급 후 SDK를 초기화하고 카메라 모듈 연결을 설정합니다.

4

개발 시작

샘플 코드와 API 레퍼런스를 참고하여 카메라 기능을 구현합니다.

// C++ - 카메라 초기화 예시 #include <lginnotek/camera_sdk.h> int main() { // SDK 초기화 LGI::CameraSDK sdk("YOUR_API_KEY"); sdk.initialize(); // 카메라 모듈 연결 및 스트림 시작 auto camera = sdk.openCamera("module_id_0"); camera->startPreview(LGI::Resolution::FHD_1080P, 60); // 이미지 캡처 camera->capture([](LGI::Image img) { img.save("output.jpg"); }); return 0; }
