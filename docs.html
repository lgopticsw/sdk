LG Innotek Camera Module SDK 문서

이 문서는 LG Innotek 카메라 모듈 SDK의 설치, 설정, API 사용 방법 및 개발 가이드를 안내합니다. SDK를 통해 LG Innotek의 카메라 모듈을 Android, iOS, Linux, Windows 및 임베디드 환경에서 손쉽게 통합할 수 있습니다.

시스템 요구사항

지원 운영체제

  • Android: API Level 28 (Android 9 Pie) 이상
  • iOS / iPadOS: iOS 14.0 이상
  • Linux: Ubuntu 20.04 LTS 이상, Yocto Project 3.1+
  • Windows: Windows 10 (1903 이상) / Windows 11
  • Embedded: FreeRTOS 10.x, ThreadX 6.x

개발 도구

  • C/C++ 컴파일러: GCC 9+ 또는 Clang 10+ (MSVC 2019+ for Windows)
  • CMake 3.16 이상
  • Android Studio 4.2+ (Android 개발 시)
  • Xcode 13+ (iOS 개발 시)

하드웨어 요구사항

  • LG Innotek 카메라 모듈 (지원 모델 목록 참조)
  • USB 3.0 또는 MIPI CSI-2 인터페이스
  • 최소 RAM: 2GB (실시간 4K 처리 시 8GB 권장)

설치 가이드

SDK 사용을 위해 API 키가 필요합니다. LG Innotek 개발자 포털에서 API 키를 발급받으세요.

Android (Gradle)

// build.gradle (app) dependencies { implementation 'com.lginnotek:camera-sdk:3.2.1' }

iOS (Swift Package Manager)

// Package.swift dependencies: [ .package(url: "https://github.com/lginnotek/camera-sdk-ios", from: "3.2.1") ]

Linux / Embedded (CMake)

# CMakeLists.txt find_package(LGICameraSDK 3.2 REQUIRED) target_link_libraries(MyApp PRIVATE LGICameraSDK::core)

퀵스타트

아래 예제는 카메라 모듈을 초기화하고 프리뷰를 시작한 후 이미지를 캡처하는 최소한의 예제 코드입니다.

C++ 예제

#include <lginnotek/camera_sdk.h> #include <iostream> int main() { // 1. SDK 초기화 LGI::CameraSDK sdk("YOUR_API_KEY"); if (!sdk.initialize()) { std::cerr << "SDK 초기화 실패\n"; return -1; } // 2. 카메라 모듈 열기 auto camera = sdk.openCamera("module_id_0"); // 3. 프리뷰 설정 및 시작 (FHD 60fps) LGI::CaptureConfig cfg; cfg.resolution = LGI::Resolution::FHD_1080P; cfg.fps = 60; camera->configure(cfg); camera->startPreview(); // 4. 스틸 이미지 캡처 camera->capture([](LGI::Image img) { img.save("output.jpg", LGI::ImageFormat::JPEG); std::cout << "캡처 완료: output.jpg\n"; }); camera->stopPreview(); sdk.shutdown(); return 0; }

Android (Kotlin) 예제

import com.lginnotek.camerasdk.CameraSDK import com.lginnotek.camerasdk.Resolution val sdk = CameraSDK("YOUR_API_KEY") sdk.initialize(applicationContext) val camera = sdk.openCamera("module_id_0") camera.startPreview(Resolution.UHD_4K, fps = 30) camera.capture { image -> image.saveTo(outputFile) }

API 레퍼런스

SDK의 주요 API 목록입니다. 각 API에 대한 상세 설명은 아래를 참조하세요.

CameraSDK 클래스

메서드 반환 타입 설명
initialize() bool SDK를 초기화합니다. API 키 유효성을 검증하고 드라이버를 로드합니다.
openCamera(id) Camera* 지정된 ID의 카메라 모듈을 열고 Camera 객체를 반환합니다.
listDevices() vector<DeviceInfo> 연결된 모든 카메라 모듈의 장치 정보 목록을 반환합니다.
shutdown() void SDK 리소스를 해제하고 모든 카메라 연결을 종료합니다.

캡처 API

메서드 반환 타입 설명
configure(CaptureConfig) bool 해상도, FPS, 포맷 등 캡처 파라미터를 설정합니다.
startPreview() void 카메라 프리뷰 스트림을 시작합니다.
stopPreview() void 카메라 프리뷰 스트림을 중지합니다.
capture(callback) void 스틸 이미지를 캡처하고 콜백으로 Image 객체를 전달합니다.
startRecording(path) bool 지정 경로에 동영상 녹화를 시작합니다.
stopRecording() void 동영상 녹화를 중지합니다.

ISP 제어 API

파라미터 타입 범위 설명
setExposure(value) float -4.0 ~ +4.0 EV 노출 보정 값을 설정합니다.
setISO(value) int 50 ~ 204800 ISO 감도를 설정합니다.
setWhiteBalance(mode) WBMode AUTO, DAYLIGHT, CLOUDY, ... 화이트 밸런스 모드를 설정합니다.
setFocusMode(mode) FocusMode AUTO, MANUAL, CONTINUOUS 초점 모드를 설정합니다.
setZoom(ratio) float 1.0 ~ 10.0x 디지털/광학 줌 배율을 설정합니다.
